1. 서버(Server)
    1) 웹에서 서비스를 제공하는 컴퓨터 시스템 
    2) 서비스를 제공하는 웹 애플리케이션은 서버 컴퓨터에 파일의 형태로 저장되어 있음

2. 웹 서버(Web Server)
    1) 사용자로부터 HTTP를 통해 요청을 받거나, 웹 컨테이너가 전달해준 결과물을 정적인 페이지로 생성하여 
       사용자에게 응답해주는 소프트웨어 
    2) 웹 페이지는 주로 HTML, CSS, JS 등으로 구성됨

3. 웹 컨테이너(Web Container)
    1) 웹 서버가 전송해준 요청을 기초로 동적인 페이지를 생성하여 웹 서버로 돌려줌
    2) 동적인 페이지 
        -사용자마다 다른 결과로 응답할 수 있기 때문임
        -간단한 예로 로그인이 있음 
            -로그인한 아이디를 보고 각자에 맞는 페이지를 구성해 보내줌

4. WAS(Web Application Server)
    1) 웹 애플리케이션이 실행될 수 있는 환경을 제공하는 소프트웨어 
        -컴퓨터에서 운영체제와 비슷한 역할을 하는 소프트웨어 
    2) 웹 서버와 웹 컨테이너를 포함한 개념임
    3) 예: 톰캣(Tomcat), 웹로직, 웹스피어 등
